# Support Tiers

## Overview

Citadel is available under MIT/Apache-2.0 open source license. Commercial support tiers provide additional guarantees and services.

---

## Open Source (Free)

**License:** MIT OR Apache-2.0

**What you get:**
- Full source code
- All features
- GitHub Issues (best effort)
- Public documentation

**What you don't get:**
- Guaranteed response times
- Security notifications
- Version support commitments
- Direct contact

**Best for:**
- Evaluation
- Personal projects
- Early-stage startups
- Non-critical systems

---

### Upgrade Support
- Changelog review calls (quarterly)
- Migration assistance for major versions
- Compatibility testing guidance

### Documentation

- Internal design documents
- Integration guides for common stacks

**Pricing:**

## FAQ

**Q: Can I use the open source version in production?**

Yes. The license permits commercial use. You're responsible for your own support.

**Q: What if I need just one Pro feature?**

Contact us. We can discuss custom arrangements.

**Q: Do support tiers include indemnification?**

No. Indemnification requires a separate agreement. Contact us to discuss.

**Q: Can I get support without a subscription?**

Yes. We offer incident-based support at $500/hour (4-hour minimum). This does not include SLA or security notifications.

**Q: What's the cancellation policy?**

- Pro: Cancel anytime, prorated refund
- Enterprise: 90-day notice, no prorated refund
